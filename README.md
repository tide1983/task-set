# Task Set

## Описание
Решение задачи "Set" из домашнего задания к лекции «Контейнеры».

## Реализовано
- Класс `Team` с использованием `Set` для хранения персонажей
- Метод `add()` - добавление персонажа с проверкой на дублирование
- Метод `addAll()` - добавление нескольких персонажей с игнорированием дублей
- Метод `toArray()` - конвертация Set в массив

## Технологии
- JavaScript (ES6+)
- Jest для тестирования
- Babel для транспиляции
- Webpack для сборки
- ESLint для проверки кода

## Установка и запуск
```bash
npm install
npm test          # Запуск тестов
npm run test:coverage  # Запуск тестов с покрытием
npm run build     # Сборка проекта
